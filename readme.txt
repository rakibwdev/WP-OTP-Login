== Description ==
Login with Email OTP (free) or Phone OTP (SMS API).

== Installation ==
1. Upload plugin
2. Activate
3. Go to Settings → WP OTP Login
4. Use shortcode [wp_otp_login]

== Features ==
- Email OTP (Free)
- Phone OTP (SMS)
- Auto user registration
- Secure AJAX
