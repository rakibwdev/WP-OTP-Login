== Description ==
WP OTP Login allows users to log in securely using One-Time Passwords (OTP) via Email or Phone (SMS). Eliminate passwords and improve security with a fast and user-friendly authentication system.

Features include email OTP (free), phone OTP via SMS API, automatic user registration, and AJAX-based secure login.

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings → WP OTP Login**
4. Configure your preferred login method
5. Use shortcode `[wp_otp_login]` to display the login form

== Features ==
- Email OTP login (Free)
- Phone OTP login via SMS API
- Passwordless authentication
- Auto user registration
- Secure AJAX-based system
- Admin control (Email / Phone / Both)
- Easy shortcode integration

== FAQ ==

= Does this plugin require an SMS API? =
No, SMS API is only required for phone OTP login.

= Can I use only email OTP? =
Yes, you can enable email-only login from settings.

= Does it create users automatically? =
Yes, new users are automatically registered after OTP verification.
