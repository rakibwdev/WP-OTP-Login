=== WP OTP Login ===
Tags: otp, login, mobile login
Requires at least: 5.0
Tested up to: 6.5
Stable tag: 1.0

== Description ==
Login with mobile number using OTP. Supports Firebase and SMS API.

== Features ==
- Firebase OTP (free)
- SMS OTP (custom API)
- Phone login system

== Usage ==
Use shortcode:
[wp_otp_login]
